# Golf (2D) — Three.js School Project

Play at: [https://k3vlar159.github.io/golf/](https://k3vlar159.github.io/golf/)

A small **2D golf game** built as a **school project** using **JavaScript** and **Three.js**.  
The goal is simple: aim, set power, and take as few strokes as possible to get the ball into the hole while avoiding water pools.

## Tech Stack

- **JavaScript** (main codebase)
- **Three.js** (rendering)
- **GLSL** (small amount of shader code)

## Features

- Aim + power-based shooting
- Basic physics-style ball movement (momentum / friction-style slowdown)
- Use "booster" and "jumper" power-ups to help you reach the hole
- Gravity slider
- **Procedurally generated terrain**, including **water** and **sand** hazards/features
